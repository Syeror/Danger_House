Как использовать:

Добавьте скрипт к объекту игрока.
Укажите префаб объекта, который будет подбираться, в поле itemPrefab.
Укажите расстояние, на котором можно подобрать предмет, в поле pickupDistance.
Добавьте компонент PickableItem к объектам, которые будут подбираться.
Объяснение:

Скрипт ItemPickup находится на объекте игрока.
Метод Update вызывается каждый кадр.
В методе Update используется метод Physics.OverlapSphere для поиска всех коллайдеров в радиусе pickupDistance от игрока.
Для каждого найденного коллайдера проверяется, есть ли на нем компонент PickableItem.
Если компонент PickableItem найден, вызывается метод PickItem для подбора предмета.
Метод PickItem уничтожает объект, который был подобран, создает новый объект префаба в позиции игрока и выполняет действия, связанные с подбором предмета.
Дополнительно:

Вы можете добавить дополнительные проверки в метод PickItem, чтобы, например, разрешить подбирать предметы только при нажатии кнопки.
Вы можете изменить логику создания нового объекта префаба, чтобы, например, добавить его в инвентарь игрока.
Вы можете добавить визуальные и звуковые эффекты при подборе предмета.
